🚀 NumPy Data Analysis Project – Student Performance Dataset

I recently completed a data analysis project using NumPy, where I analyzed a dataset containing student marks across multiple subjects.

The goal of this project was to understand how NumPy can be used for efficient numerical computations and statistical analysis on structured datasets.

📊 Key tasks performed in this project:

• Converted dataset into NumPy arrays for numerical analysis
• Calculated mean, median, and standard deviation for subject performance
• Identified highest and lowest scores across subjects
• Calculated total marks and average marks for each student
• Identified top-performing students and students needing improvement
• Performed statistical analysis using NumPy functions

📈 Key Insights:

• Identified subjects with the highest and lowest average marks
• Analyzed score variations across different subjects
• Identified top-performing students based on average marks
• Detected students requiring additional academic support

This project helped me strengthen my understanding of NumPy arrays and numerical data analysis, which are fundamental for data science, machine learning, and scientific computing.

💻 Tools Used:

Python | NumPy | Pandas | Jupyter Notebook
